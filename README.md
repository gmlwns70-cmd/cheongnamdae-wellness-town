# 청남대 웰니스타운 홈페이지

GitHub Pages용 정적 홈페이지입니다.

## 빠른 수정
`assets/js/config.js`에서 다음 항목을 수정하세요.
- 전화번호
- 카카오톡 상담 링크
- 지도 링크
- 인스타그램 / 유튜브
- 주소 / 이메일

## GitHub Pages 배포
1. GitHub에서 `cheongnamdae-wellness-town` 저장소 생성
2. 이 폴더 안의 파일 전체를 저장소 루트에 복사
3. GitHub Desktop에서 Commit → Push origin
4. Settings → Pages → Deploy from a branch → `main` / `(root)` → Save

## 중요
- 파일명은 영문 소문자 유지
- `assets` 폴더 구조 변경 금지
- 실제 고객 후기, 주소, 가격, 운영시간은 확인 후 교체
